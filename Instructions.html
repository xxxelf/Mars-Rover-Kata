How to Control the Mars Rover

To move Forward type: move("f",for Rover1 use r1 for Rover2 use r2)

To move Backward type: move("b",for Rover1 use r1 for Rover2 use r2)

To turn Left type: move("l",for Rover1 use r1 for Rover2 use r2)

To turn Right type: move("r",for Rover1 use r1 for Rover2 use r2)

Have fun and please don't break the Rovers ;)
